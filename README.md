# Pushdown Automata and Context-free Languages
## 基本モデル
- `pdaClass.py`
    - `[pZq]`の三組を保存するクラス: `TR`
    - 決定性PDA: `DPDA`
    - 非決定性PDA: `NPDA`
- `cfg.py`
    - 文脈自由文法: Greibach標準形: `CFG`

## 実行例
- 決定性PDA: `DPDASample1.ipynb`
- 非決定性PDA: `NPDASample1.ipynb`
- 文脈自由文法とPDA: `cfg2PDA.ipynb`
